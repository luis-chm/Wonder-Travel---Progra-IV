# Wonder Travel - Sitio Web de Agencia de Viajes

## 📝 Descripción
Sitio web responsivo de una sola página (single-page) para la agencia de viajes ficticia "Wonder Travel". El sitio está desarrollado utilizando HTML, CSS y Bootstrap, enfocado en promocionar destinos de montaña.

## 🛠️ Tecnologías Utilizadas
- HTML5
- CSS3
- Bootstrap 5.3.2
- Bootstrap Icons
- FontAwesome Icons

## 🚀 Características
- Diseño completamente responsivo
- Single-page navigation
- Formulario de reserva completo
- Animaciones y transiciones suaves
- Integración con Bootstrap Icons y FontAwesome Icons
- Optimizado para diferentes dispositivos

## 📱 Secciones
1. **Navbar**: Navegación fija con logo y enlaces a secciones
2. **Hero**: Presentación principal de la oferta de vacaciones
3. **Atracciones**: Tres destinos principales disponibles
4. **Tours**: Tres paquetes turísticos detallados
5. **Formulario de Reserva**: Sistema completo de reservación
6. **Footer**: Información de contacto y enlaces importantes

## 💻 Instalación y Uso
1. Clonar o descargar el repositorio
2. Abrir `index.html` en un navegador web moderno
3. No requiere instalación adicional

## 🌐 Compatibilidad
